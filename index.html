Kanban Board

The kanban board is a web application that allows one to organize an manage their work in a visual and intuitive maner.

Users can add/delete tasks and move them from left to right by dragging and dropping the tasks in the columns.

Read more
See Live Github
See Live Github

Project Details:

Challenge:

To build an application where I could organize my work and daily tasks. The app is inspired by the Kanban board and should contain the following features:

  • Four status columns;
  • Drag and drop the tasks between the columns;
  • A delete button for each task separately;
  • The tasks must be saved in the local storage;
  • Responsive design.

Solution:

To build this app I have used HTML5, CSS3 and JavaScript. In order to meet all the requirements, the below concepts were applied:

  • CSS Flexbox for positioning and styling;
  • Media Queries for the responsive design;
  • HTML Drag and Drop API for the 'grab and drop' functionality;
  • Web EventListener API and Array methods for the buttons and other features;
  • DOM Manipulation;
  • Web Storage API and its methods for saving the data to local storage;

Outcome:

The final project is very reliable and useful on a daily basis. All the wanted features were implemented and the app works very well. Building this project helped me to learn new JavaScript concepts and improve my coding skills as a whole.

The user can search for any city in the world and the app will display the weather forecast for the location in that moment.

See Live Github

Project Details:

Challenge:

In order to practice my acquired JavaScript skills I decided to create a weather application with the following features:

  • When the users open the page, it loads the weather in their location;
  • A search field where the user can type the name of the city;
  • Error messages in case of empty field or not existing city name
  • Display all the weather data in the UI with a responsive design

  • CSS Flexbox and Grid

Outcome:

The application works well, and it helped me to understand better the OOP and Async/Await concepts.
